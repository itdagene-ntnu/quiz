# itdagene-quiz

Quiz app for itDAGENE. Segregated from the main project since 2014.

## Getting set up

### Dependencies

* Virtualenv
* bower
* PostgreSQL

#### Clone the repo

    $ git clone https://github.com/itdagene-ntnu/quiz.git

#### Set up a virtual environment

    $ cd quiz
    $ virtualenv venv
    $ source venv/bin/activate

#### Run make-target setup

    $ make setup
